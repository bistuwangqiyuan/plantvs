# 🌻 植物大战僵尸 - 竖屏网页版 🧟

一个使用原生HTML5、CSS和JavaScript开发的竖屏版植物大战僵尸游戏。

## 🎮 游戏特色

- ✅ **竖屏优化**: 专为移动设备竖屏模式设计
- ✅ **纯前端实现**: 无需后端服务器，完全在浏览器中运行
- ✅ **3个关卡**: 难度递进的3个游戏关卡
- ✅ **5种植物**: 向日葵、豌豆射手、坚果墙、樱桃炸弹、寒冰射手
- ✅ **3种僵尸**: 普通僵尸、路障僵尸、铁桶僵尸
- ✅ **触控友好**: 优化的触控操作体验
- ✅ **进度保存**: 自动保存游戏进度到本地
- ✅ **响应式设计**: 适配移动设备和桌面浏览器

### 本地运行

1. **克隆仓库**
   ```bash
   git clone <repository-url>
   cd plantvs
   ```

2. **启动本地服务器**
   
   使用Python:
   ```bash
   python -m http.server 8000
   ```
   
   或使用Node.js:
   ```bash
   npx serve -p 8000
   ```

3. **打开浏览器**
   
   访问 `http://localhost:8000`

## 🎯 游戏玩法

### 基本规则

1. **收集阳光**: 点击天空掉落的阳光或向日葵生产的阳光
2. **种植植物**: 使用阳光种植植物来抵御僵尸
3. **防御僵尸**: 阻止僵尸到达屏幕左侧边界
4. **获得胜利**: 消灭所有波次的僵尸

### 键盘快捷键 ⌨️

| 按键 | 功能 |
|------|------|
| **空格键** | 暂停/继续游戏 |
| **ESC键** | 打开暂停菜单 / 返回主菜单 |
| **R键** | 重新开始关卡（暂停时） |

### 植物介绍

| 植物 | 阳光消耗 | 冷却时间 | 特性 |
|------|---------|---------|------|
| 🌻 向日葵 | 50 | 7.5秒 | 每24秒生产25阳光 |
| 🌰 豌豆射手 | 100 | 7.5秒 | 发射豌豆攻击僵尸，伤害20 |
| 🥜 坚果墙 | 50 | 30秒 | 高血量防御，4000血量 |
| 💣 樱桃炸弹 | 150 | 50秒 | 范围爆炸，伤害1800 |
| ❄️ 寒冰射手 | 175 | 7.5秒 | 发射寒冰豌豆，减速50% |

### 僵尸类型

| 僵尸 | 血量 | 特性 |
|------|------|------|
| 🧟 普通僵尸 | 200 | 基础僵尸 |
| 🧟 路障僵尸 | 640 | 戴路障，血量更高 |
| 🧟 铁桶僵尸 | 1370 | 戴铁桶，血量极高 |

## 🛠️ 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和布局
- **JavaScript ES6+**: 游戏逻辑
- **Canvas API**: 游戏渲染
- **localStorage**: 数据持久化
- **Bootstrap 5**: UI组件 (CDN)

## 🎨 开发特性

- 原生JavaScript，无框架依赖
- 扁平文件结构，易于理解
- Canvas 2D渲染，高性能
- 对象池模式，优化内存
- 固定时间步长游戏循环
- 响应式设计，自适应屏幕

## 🚢 部署到Netlify

1. **连接仓库**
   - 登录 [Netlify](https://netlify.com)
   - 选择 "Add new site" → "Import an existing project"
   - 连接GitHub/GitLab仓库

2. **配置部署**
   - Build command: 留空（静态站点）
   - Publish directory: `.` (根目录)
   - 点击 "Deploy site"

3. **自定义域名** (可选)
   - 在Netlify设置中配置自定义域名
   - 自动启用HTTPS
